## ✨ Features

### 🎮 Games Included

| Game | Description | Difficulty | Rewards |
|------|-------------|:----------:|:-------:|
| 🔢 **Mini Sudoku 3×3** | Fill the grid with numbers 1-9 without repetition | ⭐⭐ | 50-150 coins |
| 🧮 **Math Quest** | Solve math problems against the clock | ⭐-⭐⭐⭐ | 20-100 coins |
| 🃏 **Memory Match** | Find matching emoji pairs before time runs out | ⭐⭐ | 30-120 coins |
| 🔮 **Pattern Master** | Identify and complete number sequences | ⭐⭐⭐ | 40-150 coins |
| 📝 **Word Scramble** | Unscramble letters to discover hidden words | ⭐⭐ | 25-80 coins |

### 🏆 Reward System

- 💰 **Coins** – Earn coins by completing games successfully  
- ⭐ **Stars** – Collect stars for exceptional performance  
- 📈 **Levels** – Level up every 500 coins earned  
- 🔥 **Daily Streak** – Bonus rewards for consecutive daily plays  
- 🎁 **Prize Shop** – Redeem coins for badges, avatars, and power-ups  

### 🎨 Kid-Friendly Design

- 🌈 Colorful and attractive UI designed for kids  
- ✨ Smooth animations and visual feedback  
- 🎵 Engaging interactions with confetti celebrations  
- 📱 Fully responsive – works on desktop, tablet, and mobile  
- 💾 Progress saved automatically using localStorage

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required

### Installation

1. Download or clone the project
2. Open the project folder
3. Open `index.html` in your browser
4. Start playing 🎮
